# Discord Alert

**Discord Alert** — это современное десктопное приложение для управления Discord вебхуками. Дизайн выполнен в стиле Android Dark с использованием PyQt6.

---

> [!WARNING]
> Программа в постоянном тестировании и обновлении поэтому возможна встреча багов

## ✨ Основные функции

*   **⚡ Мгновенная отправка**: Сообщения улетают в Discord за миллисекунды.
*   **🎨 Embed-конструктор**: Создавайте красивые карточки внутри сообщений.
*   **🌈 Кастомные цвета**: У каждого Embed-блока своя палитра выбора цвета.
*   **📱 Android Style UI**: Темная тема и плавные анимации появления блоков.
*   **🧩 Динамика**: Добавляйте и удаляйте неограниченное количество блоков.

Программа имеет единый .exe файл для удобства использования в различных целях - то есть, для планирования
кастомных сообщений от лица WebHook

---

## 📖 Как пользоваться

1. Вставьте **Webhook URL** из настроек канала.
2. Напишите текст в поле **Message**.
3. Нажмите **+ ДОБАВИТЬ EMBED**, заполните данные и выберите цвет.
4. Жмите **ОТПРАВИТЬ** — кнопка станет зеленой при успехе.
